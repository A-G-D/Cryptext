The following text files are optionally used by the Cryptext.exe program. They
contain the properties of the layout of the application's window. You can edit
them to change the look of the application's window. Deleting these files will
cause the program to use its default aesthetics.

- Window.txt
- MainPage.txt
- TextFilesPage.txt
- TextEditingPage.txt
- TransaltionEditingPage.txt
